Line Charts

Display HealthCheck result or AppStats data as a line chart.


Widget Url

Result based

This provides a linechart based on the Result value from HealthCheck/AppStat data.
  • http://hostname/geckoboard/linechart/check/outcome/
  • where...
    • check is the check id to target
    • outcome is the result, values are "success", "failure", "any" (for AppStats always use "any")

Count based
